# Notice

This repository contains the public LBP3000-to-LBP2900 patcher only.

It includes:

- `captdriver`, under its upstream license in `third_party/captdriver/COPYING`.
- Local macOS package scripts for installing the open CAPT raster filter and LBP2900 PPD.

Canon's proprietary StatusMonitor resources and the earlier clean reinstall package are intentionally not included in the public repository.
